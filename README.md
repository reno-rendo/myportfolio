## 🚀 Highlights

- Built with a fast, optimized React + Vite setup.
- Fully responsive layout that works cleanly on all devices.
- Modular component structure for easy updates and scaling.
- Smooth animations that enhance navigation without slowing the site.
- Deployed on Netlify with a production-grade build.

## 🔧 Running Locally

    cd portfolio
    npm install
    npm run dev

---

## 📦 Production Build

    npm run build
    npm run preview
